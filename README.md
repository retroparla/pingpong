Este repositorio recoge todo el material utilizado para la recreación del videojuego ping-pong de 1977 que pudo verse en RetroParla 2025, incluyendo:

-El capítulo "XV. Juego de ping-pong en el 8080" del libro "Microprocesadores II" de E. Muñoz Merino donde se documenta tanto el hardware como el software del juego.<br/>
-El proyecto en Kicad describiendo el hardware de la recreación.<br/>
-El código fuente del juego y su versión ya ensamblada.<br/>

El proyecto original fue creado en la E.T.S.I de Telecomunicaciones de la Universidad Politécnica de Madrid por Rafael Martínez Jiménez y Juan Miguel Santos Suárez, y todo el proceso para replicarlo está contado en el artículo <a href="https://www.retroparla.com/index.php/2025/02/recreando-el-videojuego-espanol-mas-antiguo/">Recreando el videojuego español más antiguo</a> en la web de la asociación.
